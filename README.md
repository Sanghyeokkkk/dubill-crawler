# 더빌(TheBill) 입금내역 자동 수집기

NICE정보통신 **더빌 CMS**(`thebill.co.kr`)의 가상계좌 입금내역을 자동으로 받아
**구글 시트**에 적재하는 프로그램.

```
로그인(2FA 반자동) → 입금내역조회 → 엑셀 다운로드 → 파싱
→ 구글시트 2곳 적재 (원본 탭 + OPS 관리 탭), 중복 자동 제거
```

## 빠른 시작 (이미 설정된 PC)
- 바탕화면 **"더빌 입금내역 수집"** 바로가기 더블클릭, 또는
- `더빌_입금내역_수집.bat` 더블클릭
- 크롬이 뜨면 로그인(필요시 2차 인증)만 하면 나머지는 자동

## 새 PC에 설치
→ **[SETUP.md](SETUP.md)** 참고 (Python·Chrome 설치 → `setup.bat` → 비밀정보 복사)

## 주요 파일
| 파일 | 역할 |
|------|------|
| `dubill_crawler.py` | 메인: 로그인→다운로드→파싱→배포 |
| `sheets_sync.py` | "더빌_입금내역" 탭(원본)에 적재 |
| `ops_sync.py` | "OPS 서브테스트" 탭에 지점/호수 분리 적재 |
| `config.py` | 설정 (로그인·시트URL 등) ⚠️ 비밀, git 제외 |
| `config.example.py` | 설정 템플릿 |
| `service_account.json` | 구글 접근 키 ⚠️ 비밀, git 제외 |
| `setup.bat` | 새 PC용 설치 스크립트 |
| `더빌_입금내역_수집.bat` | 수동 실행 |
| `dubill_schedule_run.bat` | 매일 자동 실행용 (스케줄러) |

## 동작 방식 요약
- 더빌은 2단계 인증이 있어 **반자동**: ID/PW는 자동 입력, 인증번호만 사람이 처리
  (`chrome_profile` 폴더에 세션을 기억해 매번 인증이 뜨지는 않음)
- 더빌 `고객명`(예: `세종대점305`)을 **지점명/호수로 분리**해 OPS 탭에 기재
- 중복은 (지점·호수·입금일·금액·입금자) 기준으로 자동 제거 → 매일/여러 번 돌려도 안전
- OPS 탭은 **gid(고유ID)로 찾음** → 탭 이름을 바꿔도 안 깨짐

## 설정값 (config.py)
- `THEBILL_ID/PW`, `MERCHANT_ID` : 더빌 로그인/가맹점
- `DAYS_AGO` : 최근 며칠 조회 (기본 7)
- `SPREADSHEET_URL` : 대상 구글시트
- `WORKSHEET_NAME` : 원본 탭 이름 (기본 "더빌_입금내역")
- `OPS_WORKSHEET_GID` : OPS 탭 gid (이름 변경에도 안전)

## 한계 / 메모
- 더빌 엑셀의 입금일시는 **날짜만** 제공 (시간 없음)
- OPS 탭의 실월세/입금상태/상세내용은 더빌에 없어 비워둠 (수동 또는 추후 자동화)
